# pl-genesis-mocha
Cross‑chain ERC‑4626 vaults and investor portal that turn real‑world coffee farms into on‑chain, yield‑bearing assets on Scroll, Base, and Starknet for PL_Genesis.

> Project Mocha – PL_Genesis Public Repo
> This repo contains the public code and docs for Project Mocha, a cross‑chain RWA DeFi system that tokenizes real‑world coffee farms as ERC‑4626 vaults on Scroll, Base, and Starknet, plus a Next.js investor portal used for the PL_Genesis: Frontiers of Collaboration Hackathon.
