Coffee-backed investment platform enabling users to invest in climate-resilient coffee farms through blockchain tokens.

**Full Description:**  
Project Mocha is a decentralized investment platform that tokenizes real-world coffee tree assets. Each MBT (Mocha Bean Token) represents ownership in coffee trees, providing investors with transparent, blockchain-verified proof of their agricultural investments. The platform enables users to invest in coffee farms, earn returns from harvest yields, and track their investments on-chain.

## Key Features

### 1. **Tokenized Coffee Investments**
- Each MBT represents ownership in real coffee trees
- 1 coffee tree = 4 MBT tokens
- Transparent on-chain proof of investment

### 2. **Multi-Chain Support**
- Deployed on Base Mainnet (primary)
- Also supports Scroll network
- Users can choose their preferred network

### 3. **Investment Features**
- Staking MBT tokens for annual returns
- Fixed annual yield with compounding
- 5-year investment projections
- Real-time investment tracking

### 4. **Payment Options**
- Crypto payments (ETH, WETH)
- M-PESA integration (via ElementPay)
- Card payments (coming soon)

### 5. **Social Login**
- Email/Google wallet creation
- Automatic wallet generation on Base network
- Seamless onboarding experience

## How to Submit

### Step 1: Visit Base Ecosystem Page

**Option A: Base.org**
1. Go to: https://base.org/ecosystem
2. Look for "Submit Your Project" or "Add Your dApp" button
3. Fill out the submission form

**Option B: Coinbase Ecosystem**
1. Go to: https://www.coinbase.com/ecosystem/base
2. Look for submission form or contact link
3. Submit your project details

**Option C: Direct Contact**
- Email: ecosystem@base.org (if available)
- Discord: Base Discord server (if available)
- Twitter: @base (tag them with your submission)

### Step 2: Fill Out Submission Form

Use the information provided above to fill out:
- Project name: **Project Mocha Investor Portal**
- Description: Use the full description from above
- Website: **https://portal-main.vercel.app**
- Category: **DeFi / Investment / RWA**
- Contract addresses: Use the addresses listed above
- Network: **Base Mainnet (8453)**

### Step 3: Attach Assets

Prepare these files:
1. **Logo:** High-resolution Project Mocha logo (PNG, 512x512px recommended)
2. **Screenshots:** 
   - Dashboard view
   - Investment interface
   - Network selector showing Base
   - Transaction history
3. **Video Demo:** (Optional) Short video showing the app

### Step 4: Submit and Follow Up

1. Submit the form
2. Wait for Base team review (typically 1-2 weeks)
3. Follow up if no response after 2 weeks
4. Once approved, your app will appear in Base ecosystem directory
